# Hero Image Placement Guide

## Where to Place Your Image

### File Location
Save your generated image as:
```
content/assets/images/hero-content.jpg
```

### Full Path
```
/Users/gastondana/Portfolio/content/assets/images/hero-content.jpg
```

## What This Image Will Replace

Your image will appear in **TWO places**:

### 1. Hero Section (Top of Content Page)
- The grey/silver square box with the play button icon
- Located right below the "Content Creation" title
- Dimensions: Approximately 400x400px display (but use 800x800px for retina)
- This is the main visual on your content creation landing page

### 2. Main Portfolio Discovery Card
- When people visit your main tech portfolio
- The content creation card that links to this page
- Shows a preview of your content work

## Image Specifications

**For Best Results:**
- **Primary Image**: 1200x800px (16:9 ratio) - for hero section background
- **Square Version**: 800x800px (1:1 ratio) - for the visual box
- **Format**: JPG (smaller file size) or PNG (if you need transparency)
- **Quality**: 85-90% compression for web optimization

## Quick Steps

1. Generate your image using Prompt #2 (Tech Meets Creativity)
2. Save it as `hero-content.jpg`
3. Place it in: `content/assets/images/`
4. Refresh your browser - it will automatically load!
