## Architecture Highlights

### Design Philosophy
- **No Framework Bloat** - Pure vanilla JS frontend for maximum performance
- **Progressive Enhancement** - Works without JavaScript, enhanced with it
- **Mobile-First** - Designed for mobile, scaled up for desktop
- **API-First** - Clean separation between frontend and backend

### Technical Decisions
- **Vanilla JS over React/Vue** - Faster load times, smaller bundle size
- **Three.js for 3D** - Industry-standard WebGL library
- **Flask over Django** - Lightweight, flexible, perfect for APIs
- **ChromaDB for Vectors** - Fast, embeddable, Python-native
- **LangChain for AI** - Modular, extensible, production-ready

### Performance Metrics
- **First Contentful Paint** - < 1.5s
- **Time to Interactive** - < 3.5s
- **Lighthouse Score** - 95+ across all categories
- **Bundle Size** - < 200KB (gzipped)
